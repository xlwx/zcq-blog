Mac下sublime text3使用Win键盘快捷键总结

2017-08-14

#Mac下sublime text3使用Win键盘快捷键总结

最重要的一点是，之前mac键盘下快捷键的 command 全部换成 win

文本编辑

  1. win + c／v ： 复制粘贴
  2. win + z ： 撤销修改
  3. win + enter ：在行后插入新行
  4. win + enter + shift： 在行前插入新行
  5. F6 ： 单词拼写检查
  6. win + [] : 左右缩进

选择

  1. win + D： 选择单词
  2. win + L： 选择当前行，重复可选择下一行
  3. win + 方向键： 移动光标
  4. shift + 方向键：多行选择

查找和替换

  1. ctrl + g ： 跳到某一行
  2. ctrl + p： 输入@， 查找某个函数

界面编辑

  1. win + shift + p： 打开命令面板
  2. win + k b： 打开／关闭侧边栏
  3. ctrl + tab ：切换页面
