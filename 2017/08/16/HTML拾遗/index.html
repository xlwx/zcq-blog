HTML 拾遗

作为一个前端工程师，自己的 css 和 html 技能是在是很捉急，都是实习的时候缺哪补哪，东拼西凑出来的。这篇文章总结一下 html 中被自己忽略的地方。

Tag a 的使用

  • 超链接跳转

这是 tag a 最简单的用法。target=”_blank”指定打开新页面并跳转。

<a href="xxx" target="_blank">jump</a>
  • 页面内跳转

通过标签 id 跳转。

<div id="top">Top</div>
<a href="#top">Go to top</a>
  1. 发送邮件

格式有点类似带参数的url。Mailto必须有，其他参数可选。

<a href="Mailto:1234567@163.comCC=ghsau@163.com&BCC=ghsau@163.com&Subject=Hello&Body=你好">
    给我发邮件</a>

DOCTYPE决定浏览器渲染模式

HTML5提供的 <!DOCTYPE HTML> 是标准模式, 记得在最开头声明。

Repaint 和 Reflow

我工作时犯过的错误：

  1. 绝对不要使用table
  2. 避免在 css 中使用运算式 （sass是编译后生成css，因此可以使用运算式）
  3. Avoid setting multiple inline styles：使用常用的class的方式进行设置样式，以避免设置样式时访问DOM的低效率
  4. 不要一条一条地修改 DOM 的样式。与其这样，还不如预先定义好 css 的 class，然后修改 DOM 的 className

  5. 使用cloneNode在外部更新节点然后再通过replace与原始节点互换。

    var orig = document.getElementById('container');
var clone = orig.cloneNode(true);
var list = ['foo', 'bar', 'baz'];
var content;
for (var i = 0; i < list.length; i++) {
   content = document.createTextNode(list[i]);
   clone.appendChild(content);
}
orig.parentNode.replaceChild(clone, orig);

绘图

HTML5 提供了两种绘图方式：

  1. <canvas> : 通过 javascript 动态绘图，不支持事件处理，适合图像密集型游戏。
  2. <svg>: 绘制矢量图，依赖分辨率。

inline 和 block 的区别

block会新起一行，适用于大块的元素，可以包含inline元素 例如 <div>,<p>,<h1>。inline不会新起一行，而是依照现有的布局，紧密依次排列，只关心元素的宽度,例如<span>。inline元素可以包含其他inline元素，不能包含block元素。
